# Netflix Content Trend Analysis

## Overview
Exploratory data analysis of the Netflix dataset to understand content distribution, growth trends, and rating patterns.

## Tools Used
Python
Pandas
Matplotlib
Seaborn

## Analysis Performed
- Missing value handling
- Year-wise content addition trends
- Country distribution of content
- Movie vs TV show comparison

## Insights
The dataset reveals significant growth in Netflix content over recent years and highlights dominant production regions.
